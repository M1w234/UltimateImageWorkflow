## Getting Started

1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
4. Add your API keys in Settings
5. Start creating!

## API Keys Setup

You'll need API keys from:
- [Google AI Studio](https://makersuite.google.com/app/apikey) for Gemini
- [OpenAI Platform](https://platform.openai.com/api-keys) for Vision models
- [Kling AI](https://klingai.com) for video generation

All keys are stored locally in your browser only.
